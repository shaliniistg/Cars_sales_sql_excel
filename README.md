# Cars Sales Reporting Dashboard | SQL + Excel

End-to-end car sales reporting dashboard using MySQL, Excel Pivot Tables, charts, and executive insights.

## Project Workflow
- Extracted KPIs using MySQL queries
- Exported SQL outputs into Excel
- Performed Pivot Table analysis
- Built charts and KPI visuals
- Designed an executive dashboard
- Added business insights summary

## Key Insights
- Mercedes recorded the highest average retail price
- Germany had the highest car model diversity
- Kia showed the widest price range
- Luxury vehicles dominated with 59% share

## Tools Used
- MySQL
- MS Excel
- Pivot Tables
- Pivot Charts
- Dashboarding
